# サイトドック — 実測 Webセキュリティ診断プラットフォーム

ドメインを入力するだけで、外部から観測できる4領域を **実際にスキャン** してスコアを返す、非侵入型の無償クイック診断ツールです。体験版とは違い、結果はすべて対象ドメインの実測値です。

## 診断する4領域

| 領域 | 実測している内容 |
|------|------------------|
| SSL/TLS証明書 | 証明書の有効期限・発行元、交渉プロトコル、TLS1.0/1.1の受け入れ有無、HTTP→HTTPSリダイレクト |
| セキュリティヘッダー | HSTS / CSP / X-Frame-Options / X-Content-Type-Options / Referrer-Policy / Permissions-Policy の有無、サーバ情報の露出 |
| DNS認証（メール） | SPF（-all/~all）、DMARC（p=ポリシー）、DKIM（代表的セレクタの照会） |
| フィッシング偽装 | 視覚的に紛らわしい類似ドメインの実登録・稼働の有無 |

## 起動方法

1. **`start.bat` をダブルクリック**（または PowerShell で `python server.py`）
2. ブラウザで **http://127.0.0.1:8787/** を開く
3. ドメインを入力して「スキャン開始」

初回のみ、依存パッケージのインストールが必要です:

```
python -m pip install dnspython
```

## ファイル構成

```
sentinel-audit/
├── server.py        診断サーバー（HTTP + /api/scan）
├── scanner.py       実測診断エンジン（標準ライブラリ + dnspython）
├── public/
│   ├── index.html   ホーム（営業LP）… "/"  で配信
│   └── scan.html    実測診断ツール（入力→スキャン→結果）… "/scan" で配信
├── start.bat        起動スクリプト
└── README.md
```

サイト構成（同じサーバーが両方を配信）:

| URL | ページ |
|-----|--------|
| `/` | 営業LP（ホーム）。「無償で今すぐ診断」ボタンから `/scan` へ |
| `/scan` | 実測診断ツール。ロゴクリックで `/` に戻る |
| `/api/scan?domain=...` | 診断API（JSON） |

## 技術メモ

- 依存は **dnspython** のみ。SSL・HTTP・並列処理はPython標準ライブラリ。
- 1ドメインの診断は通常 **0.5〜3秒**（4領域を並列実行）。
- 同一IPからのリクエストは60秒あたり10回に制限（`server.py` の `_rate_ok`）。
- 実在しないドメインは404で弾き、誤ったスコアを出しません。

## 診断の性質（クライアントへの説明用）

- 本ツールは **非侵入型**。対象サーバーへ侵入・負荷をかけません。公開情報のみを観測します。
- ペネトレーションテスト（侵入テスト）は **含みません**。
- DKIMは代表的なセレクタのみ照会するため、未検出でも設定済みの場合があります。
- 類似ドメインの検出は「登録・稼働の有無」を示すもので、悪性を断定しません。
- 本結果はすべての脆弱性の不存在を保証するものではありません。

## 公開・運用について

現状は `127.0.0.1`（自分のPC内）でのみ動作します。クライアントにWeb上で公開して使わせたい場合は、次のいずれかが必要です:

- **社内・商談デモ用**：この構成のまま、自分のPCで起動して画面共有／その場で入力してもらう（すぐ使えます）
- **一般公開用**：VPS（さくら/ConoHa/AWS等）に配置し、`HOST` を `0.0.0.0` に変更、リバースプロキシ（nginx）＋HTTPS化。レート制限・ログ・利用規約の整備を推奨。
